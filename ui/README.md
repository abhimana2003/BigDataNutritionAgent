# UI prototype

This folder contains a simple static HTML/CSS prototype of the UI design you attached.

To preview locally:

```bash
cd ui
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Navigation:

- The prototype has three pages (Goal Aware Planning, Adaptive Planning, Personalized Choice).
- Use the arrow buttons in the header or the left/right keyboard arrows to move between pages.
- Clicking the CTA inside a phone card will advance to the next page.
