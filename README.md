# BigData
## Setup

1. Create a Groq API key: https://console.groq.com/keys

2. Copy the example env file

    ```bash
    cp .env.example .env
    ```

3. Add your key

    ```env
    GROQ_API_KEY=your_groq_key_here
    ```

4. Run the project in the terminal

    ```bash
    ./run_project.sh
    ```
